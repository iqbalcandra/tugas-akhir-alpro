# tugas-akhir-alpro

Mini Project - Sistem Manajemen Nilai

Nama: SETYA IQBAL PUTRA CANDRA NRP: 251-111-003 PRODI : S1 INFORMATIKA

--DESKRIPSI SINGKAT-- Aplikasi ini adalah sistem berbasis CLI (Command Line) sederhana untuk mengelola nilai mahasiswa. Masalah yang diselesaikan adalah digitalisasi pencatatan nilai agar guru/dosen tidak perlu mencatat manual dan dapat melakukan perubahan data dengan cepat.

Fitur

Tambah Data: Menambahkan absen, Nama, dan Nilai ke sistem.
Lihat Data : Menampilkan tabel daftar mahasiswa yang terdaftar.
Update Data: Mengubah Nama atau Nilai berdasarkan no absen yg terdaftar
Hapus Data : Menghapus record mahasiswa dari list
Validasi Input: Menggunakan try-except untuk mencegah error saat user salah input angka.
-- CARA MENJALANKAN --

Menjalankan Program: Jalankan file dengan perintah python nilai.py.
Ketik angka 1-5 sesuai menu yang ingin dipilih, lalu tekan Enter.
Pastikan saat memasukkan nilai hanya menggunakan angka bulat. Jika memasukkan huruf, sistem akan memberikan peringatan.
Pastikan Anda mengetahui no absen mahasiswa yang benar untuk melakukan tindakan in
