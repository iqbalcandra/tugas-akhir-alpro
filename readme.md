--DESKRIPSI SINGKAT--
Aplikasi ini adalah sistem berbasis CLI (Command Line) sederhana untuk mengelola nilai mahasiswa. Masalah yang diselesaikan adalah digitalisasi pencatatan nilai agar guru/dosen tidak perlu mencatat manual dan dapat melakukan perubahan data dengan cepat.

Fitur

1. Tambah Data: Menambahkan absen, Nama, dan Nilai ke sistem.
2. Lihat Data : Menampilkan tabel daftar mahasiswa yang terdaftar.
3. Update Data: Mengubah Nama atau Nilai berdasarkan no absen yg terdaftar
4. Hapus Data : Menghapus record mahasiswa dari list
5. Validasi Input: Menggunakan `try-except` untuk mencegah error saat user salah input angka.

---

-- CARA MENJALANKAN --

1. Menjalankan Program: Jalankan file dengan perintah `python nilai.py`.
2. Ketik angka 1-5 sesuai menu yang ingin dipilih, lalu tekan Enter.
3. Pastikan saat memasukkan nilai hanya menggunakan angka bulat. Jika memasukkan huruf, sistem akan memberikan peringatan.
4. Pastikan Anda mengetahui no absen mahasiswa yang benar untuk melakukan tindakan ini.
